# E002 — Undefined function

Function called by name is not in scope.

## Common cases

```almide
foo()                                  // foo is not defined
list.lengt(xs)                         // typo: should be list.len
my_module.something()                  // forgot to import my_module
```

## Fix

- Check spelling. The diagnostic includes a "Did you mean ...?" suggestion when one exists.
- For stdlib: ensure the module is auto-imported (Tier 1) or explicitly imported (Tier 2 — `import json`, `import math`, etc.).
- For your own modules: add `import self.<sub>` and call as `<sub>.fn()`. Or use selective import: `import self.<sub>.{fn1, fn2}`.
