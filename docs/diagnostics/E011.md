# E011 — Variable initialization issue

A variable was declared but the initialization expression has a problem (missing, ill-typed, or used before all required bindings are in scope).

## Common cases

```almide
let x =                       // E011: missing initializer
let y: Int = some_unknown     // E011: initializer references undefined name
```

## Fix

- Provide a value with `let name = <expr>`.
- Read the diagnostic — it points to what's missing on the right-hand side.
