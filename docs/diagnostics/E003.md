# E003 — Undefined variable

A name used as a value is not bound in the current scope.

## Common cases

```almide
fn f() -> Int = x                      // x is not declared
match v { ok(value) => valeu }         // typo: 'valeu' instead of 'value'
let x = y                              // y is not bound at this point
```

## Fix

- Declare with `let` or `var` before use.
- Check spelling against pattern-bound names.
- For let bindings inside expressions, ensure they're in a `Block { ... }` that contains the use.
