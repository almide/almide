# E008 — `fan` block error

A `fan { ... }` block has a structural problem (e.g., empty block, branches with incompatible result types, unsupported expression form).

## Common cases

```almide
fan { }                              // empty fan
fan { effect_fn(); pure_value }      // mixed effect / pure semantics
```

## Fix

- Provide at least one branch.
- Ensure branches return compatible result types.
- Read the diagnostic message — it includes the specific issue.
