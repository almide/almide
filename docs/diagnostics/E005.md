# E005 — Constructor argument type mismatch

A variant or record constructor was called with arguments of the wrong type.

## Common cases

```almide
type Point = Pt(Int, Int)
let p = Pt("x", "y")                  // expected Int, got String
type Box = { value: Int }
let b = Box { value: "abc" }          // record field type mismatch
```

## Fix

- Pass values matching the declared payload / field types.
- For variants with no payload, call as `MyVariant` (no parens).
- For records, ensure all required fields are present and typed correctly.
