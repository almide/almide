# E010 — Variable scope error

A variable is used outside the scope where it was bound.

## Common cases

```almide
if cond {
  let x = compute()
}
println(x)                   // E010: x not in outer scope
```

## Fix

- Bind the variable in an outer scope:
  ```almide
  let x = if cond then compute() else default
  println(x)
  ```
- Or move the use inside the block where `x` is bound.
